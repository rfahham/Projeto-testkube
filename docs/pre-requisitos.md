# Pre Requisitos para a Instalação

Preparando o ambiente para a instalação do Testkube

- Terminal
    - Iterm no MacOS
    - Terminal do WSL no Windows
    - Terninal do Linux
- Docker Desktop
- Kubernetes
- Helm
- Testkube
    - http://localhost:8080 (admin@example.com/password)
    - https://app.testkube.io/ (Acesso Github/Gitlab)
